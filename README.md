# CampbellDataCollection
A system allowing data collection from a Campbell scientific datalogger.

#Requirements:
BeagleBone Black running Debian Jessie
Internet Connection (I'm using a GSM Modem)
UART4 serial cape

#To use:
1) Clone repo
2) Run setup.sh as sudo
